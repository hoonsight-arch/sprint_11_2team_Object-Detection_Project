# 하이퍼파라미터 설정 위치

## 1. 학습 (셀 9 — `train_yolo.py` 호출)

`colab_run.ipynb`의 학습 셀에서 CLI 인자로 직접 수정.

| 인자 | 기본 | 설명 |
| --- | --- | --- |
| `--model` | `yolo11m.pt` | 모델 크기 `yolo11n/s/m/l/x.pt` |
| `--epochs` | 3 | 에폭 수 (본학습 60+ 권장) |
| `--batch` | 8 | 배치 사이즈 |
| `--imgsz` | 640 | 입력 해상도 (본학습 1280 권장) |
| `--mosaic` | 1.0 | 모자이크 증강 확률 |
| `--mixup` | 0.0 | mixup 증강 |
| `--copy_paste` | 0.0 | copy-paste 증강 |
| `--degrees` | 0.0 | 회전 (도) |
| `--flipud` | 0.0 | 상하 뒤집기 확률 |
| `--name` | `{RUN_NAME}` | 출력 폴더 이름 (셀 2의 `RUN_NAME` 자동 반영) |

`train_yolo.py`에 직접 노출되지 않은 옵션(learning rate, optimizer, **스케줄러** 등)을 바꾸려면 → 아래 2번.

## 2. Ultralytics 기본 하이퍼파라미터 (lr, 스케줄러 등)

`train_yolo.py`의 `model.train(...)` 호출에 인자를 추가하면 됩니다. 예:

```python
model.train(
    ...
    lr0=0.01,        # 초기 learning rate
    lrf=0.01,        # 최종 lr 비율 (cosine 스케줄러 끝점)
    cos_lr=True,     # cosine LR 스케줄러 사용
    optimizer='SGD', # 'SGD' | 'Adam' | 'AdamW' | 'auto'
    momentum=0.937,
    weight_decay=0.0005,
    warmup_epochs=3,
)
```

전체 옵션 목록: https://docs.ultralytics.com/modes/train/#train-settings

## 3. 추론 (셀 10 — `inference_yolo.py` 호출)

| 인자 | 기본 | 설명 |
| --- | --- | --- |
| `--conf` | 0.3 (셀에선 0.2) | 신뢰도 임계값 |
| `--iou` | 0.5 | NMS IoU |
| `--max_det` | 4 | 이미지당 최대 검출 수 |
| `--checkpoint` | `{BEST_WEIGHTS}` | 사용할 가중치 (자동) |

## 4. 데이터 / 출력 경로

`configs/default.yaml` — 그러나 노트북 셀 4에서 자동으로 덮어쓰므로 직접 수정할 일은 거의 없음.

## 요약
- **흔히 바꾸는 것**: 학습 셀의 CLI 인자 (epochs, batch, imgsz, 증강)
- **스케줄러 / lr / optimizer**: `train_yolo.py`의 `model.train(...)`에 인자 추가
- **출력 파일 이름**: 노트북 셀 2의 `RUN_NAME`, `SUBMISSION_NAME`
